# 💰 Smart Expense Tracker

> A simple, fast, and interactive command-line application to manage and analyze daily expenses using Python.

![Python](https://img.shields.io/badge/Python-3.x-blue?logo=python)
![License](https://img.shields.io/badge/License-MIT-green)
![Status](https://img.shields.io/badge/Version-v1.0.0-success)

---

## 📌 Overview

Smart Expense Tracker is a Python-based CLI application that helps users record, organize, and analyze their daily expenses. It includes expense management, analytics, and a dashboard for quick insights.

---

## ✨ Features

### 🧾 Expense Management
- Add Expense
- View Expenses
- Search Expenses
- Edit Expense
- Delete Expense

### 📊 Analytics
- Total Spending
- Category Summary
- Monthly Comparison
- Highest Expense
- Lowest Expense

### 📈 Dashboard
- Current Date
- Total Expenses
- Total Spending
- Top Spending Category

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/Manikanta-Elaprolu/smart-expense-tracker.git
```

### Navigate to the project

```bash
cd smart-expense-tracker
```

### Run the application

```bash
py src/main.py
```
